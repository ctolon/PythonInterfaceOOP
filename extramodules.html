Bases: object

For the ChoicesCompleterList package argcomplete, the TAB key is the class written for autocomplete and validation when an argument can take multiple values. By default, the argcomplete package has the ChoicesCompleter Class, which can only validate arguments that take an one value and allows autocomplete with the TAB key.

object (list) – parserargs choices object as a list
